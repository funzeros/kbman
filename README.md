### 键盘侠大作战

#### 游戏简介

> 邪恶正在悄悄靠近，快敲击你的键盘打倒怪物和敌人吧！
> 一款能够锻炼手指和释放压力的杀怪刷宝页游。

#### 项目介绍

- 技术栈 `vue3` `vite` `typescript`
- 开发环境
  - `yarn`
  - `yarn dev`
- 生产环境
  - `yarn build`
- 开发模拟生产
  - `yarn serve`
